# Product Backlog: Customer Account Microservice

Copy these into your GitHub Issues to complete tasks 5-9.

### 1. User Registration
- **Title**: Create New Customer Account
- **Role**: New User
- **Action**: Create an account with email and password
- **Value**: I can access members-only features
- **Story Points**: 8 (L)

### 2. Secure Login
- **Title**: User Authentication/Login
- **Role**: Registered User
- **Action**: Log in using credentials
- **Value**: I can access my personalized dashboard
- **Story Points**: 5 (M)

### 3. Update Profile
- **Title**: Manage Profile Information
- **Role**: Registered User
- **Action**: Update my contact details
- **Value**: My information stays accurate for shipping and billing
- **Story Points**: 3 (S)

### 4. Change Password
- **Title**: Update Account Security
- **Role**: User
- **Action**: Change my existing password
- **Value**: I can secure my account if I suspect a breach
- **Story Points**: 5 (M)

### 5. Address Management
- **Title**: Manage Multiple Shipping Addresses
- **Role**: User
- **Action**: Add, edit, or remove shipping addresses
- **Value**: I can easily ship to different locations (Home/Office)
- **Story Points**: 8 (L)

### 6. View Account History
- **Title**: Transaction/Activity History
- **Role**: User
- **Action**: View a list of my past activities and logins
- **Value**: I can monitor my account for unauthorized access
- **Story Points**: 5 (M)

### 7. Delete Account (Technical Debt)
- **Title**: GDPR Compliant Account Deletion
- **Role**: User
- **Action**: Request permanent deletion of all my data
- **Value**: My privacy is respected and my data is removed
- **Labels**: `technical debt`
- **Story Points**: 13 (XL)
