# DevOps Capstone Project: Customer Account Microservice

## Project Overview
This project is part of the DevOps Capstone, focusing on the development and deployment of a **Customer Account Microservice**. This microservice handles all user-related functionalities including account creation, authentication, and profile management.

## Features
- User Registration and Authentication
- Profile Management (Update name, email, etc.)
- Security Features (Password updates)
- Address Management
- Account Deletion (GDPR compliant)

## DevOps Practices Applied
- **Agile Planning**: Using GitHub Issues and Kanban boards.
- **Continuous Integration**: Automated testing and linting.
- **Continuous Delivery**: Automated deployment pipelines.
- **Monitoring**: Performance and health tracking.

## Repository Structure
- `user-story.md`: Template for creating standardized user stories.
- `app/`: Source code for the microservice.
- `tests/`: Automated test suite.
- `.github/`: CI/CD workflow definitions.
